# FinShark

A full stack financial data application built with React + TypeScript (frontend) and .NET (backend).

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Backend:** .NET Web API *(in progress)*
- **Database:** SQL Server *(planned)*

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
dotnet restore
dotnet run
```

> Copy `.env.example` to `.env` and fill in your values before running.
